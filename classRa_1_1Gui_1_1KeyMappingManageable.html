Detailed Description

template<typename T>
class Ra::Gui::KeyMappingManageable< T >

KeyMappingManageable decorate, typical use as a CRTP : class MyClass : public KeyMappingManageable<MyClass> { [...] it defines a static class member m_keyMappingContext, readable with getContext() This context index must be registered by the class to the KeyMappingManager with KeyMappingManager::getContext("MyClassContextName"); after a configuration file is loaded.
